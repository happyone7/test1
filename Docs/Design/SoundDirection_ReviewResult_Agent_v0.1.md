## 1. 필수 게이트 (P0)

- [x] 기존 사이버 네온 톤 폐기 + 다크 판타지 톤 재정의
- [ ] 전투 핵심 이벤트(MVP) 사운드 매핑을 실행 단위까지 정의
- [ ] AI 생성 도구/모델의 상업 이용 가능 범위 명시
- [ ] 납품 포맷/샘플레이트/루프/네이밍 기준 명시
- [ ] 구현 난이도 또는 일정 리스크 명시

판정 근거:
- 이벤트는 묶음 수준으로 언급되었으나, 이벤트별 길이/변형 수량/우선순위(P0/P1/P2)가 없음
- 라이선스는 “검증 필요”만 존재하고, 사용 가능 범위/금지 범위가 없음
- 납품 규격(포맷/샘플레이트/LUFS/네이밍)이 비어 있음
- 리스크/완화책 섹션이 없음

## 6. 수정 요청

```text
[수정요청-001]
- 우선순위: P0
- 항목: 이벤트 매핑표 상세화
- 문제: 이벤트별 길이/변형 수량/우선순위가 없어 제작 지시 불가
- 요청 수정: 14개 MVP 이벤트를 개별 행으로 분리하고, 목적/캐릭터/길이/변형수량/P0~P2를 채워 제출
- 마감: 2026-02-16

[수정요청-002]
- 우선순위: P0
- 항목: 납품 규격
- 문제: 포맷/샘플레이트/LUFS/루프/네이밍 기준 부재
- 요청 수정: BGM/SFX 각각 납품 규격 수치화 (예: 48kHz, -14~-16 LUFS, 루프 규칙, 파일명 규칙)
- 마감: 2026-02-16

[수정요청-003]
- 우선순위: P0
- 항목: 라이선스
- 문제: 상업 이용 허용 범위와 금지 조건 미정
- 요청 수정: 후보 툴별 상업 이용 조건, 저작권 귀속, 크레딧/제한사항 표로 제출
- 마감: 2026-02-16

[수정요청-004]
- 우선순위: P1
- 항목: AI 파이프라인 실무화
- 문제: 생성->보정->반입 흐름이 추상적
- 요청 수정: 툴 2개 비교표 + 선정안 1개 + 재현성 로그 포맷(모델/프롬프트/시드/버전) 제시
- 마감: 2026-02-17

[수정요청-005]
- 우선순위: P1
- 항목: 믹스 운영
- 문제: UI 우선순위/덕킹/피로도 통제 기준 부재
- 요청 수정: Mixer 버스 구조, 덕킹 규칙, 반복 피로 체크 기준 추가
- 마감: 2026-02-17
```
