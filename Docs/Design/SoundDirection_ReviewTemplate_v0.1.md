## 2. 필수 게이트 (P0)

- [ ] 기존 사이버 네온 톤이 아닌 다크 판타지 톤으로 재정의됨
- [ ] 전투 핵심 이벤트(MVP) 사운드 매핑이 누락 없이 정의됨
- [ ] AI 생성 도구/모델의 상업 이용 가능 범위가 명시됨
- [ ] 납품 포맷/샘플레이트/루프 기준이 명시됨
- [ ] 구현 난이도 또는 일정 리스크가 명시됨

판정:
- 하나라도 미충족이면 `수정요청`

## 8. 점수화 평가 (100점)

| 항목 | 배점 | 점수 |
|------|------|------|
| P0 필수 게이트 충족도 | 30 | |
| 톤/세계관 정합성 | 15 | |
| 전투 가독성/이벤트 구분력 | 15 | |
| 성장 체감/쾌감 설계 | 15 | |
| AI 파이프라인 실현 가능성 | 15 | |
| 납품 규격/운영 명확성 | 10 | |
| 총점 | 100 | |

판정 기준:
- `90~100`: 승인
- `75~89`: 조건부 승인 (P1/P2 수정)
- `0~74`: 수정요청

주의:
- P0 미충족 항목이 있으면 총점과 무관하게 `수정요청`
