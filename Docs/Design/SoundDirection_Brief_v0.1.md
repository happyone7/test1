중요: 기존 사이버 네온 기준의 사운드 방향은 모두 폐기하고, 위 2개 문서 기준으로 사운드 방향을 새로 잡아 주세요.

---

## 2. 프로젝트 사운드 방향 (요약 요청)

- 장르/톤: 다크 판타지 픽셀아트 TD
- 플레이 감각: 빠른 전투 피드백 + 성장 체감 강화
- 화면 톤: 어두운 배경 위 밝은 마법 이펙트 중심
- 사운드 역할: 가독성(정보 전달) + 쾌감(타격/성장) + 분위기(긴장/완급)

요청: 위 조건에 맞는 전체 사운드 디렉션(톤 키워드, 레퍼런스 축, 금지 요소)을 먼저 고정해 주세요.

---

## 3. 우선 산출물 요청

1. 사운드 디렉션 1차안
- BGM/SFX/UI/알림/보스 연출 톤 정의
- 레퍼런스 3~5개 + 왜 맞는지 근거

2. AI 제작 파이프라인 제안
- 사용할 생성 도구/모델 후보
- 상업 이용 라이선스/리스크 체크
- 생성 -> 보정 -> 편집 -> Unity 반입 워크플로

3. 이벤트별 사운드 매핑 시트
- `InGameCombat_v0.1.md`의 전투 피드백 이벤트 기준으로
- 필수 이벤트 우선순위(MVP/후순위) 정리

4. 납품 규격 초안
- 파일 포맷, 샘플레이트, 루프 규칙, 라우드니스 기준
- 네이밍 규칙 + 버전 규칙

---

## 4. 전투 이벤트 우선 대상 (MVP)

- 타워 공격 발사/적중/처치
- 타워 배치/합성
- 웨이브 시작/클리어
- 보스 등장/처치
- 기지 피격, 저체력 경고(HP<30%, HP<10%)
- 스테이지 클리어 팡파르
- 크리티컬 히트 강조음

---

## 5. 요청 메시지 (그대로 전달 가능)

아래 문구를 사운드 디렉터에게 전달해 주세요.

```text
사운드 방향을 기존 기준에서 전면 리셋하려고 합니다.
반드시 아래 2개 문서만 기준으로 새로 방향을 잡아 주세요.
- Docs/Design/InGameCombat_v0.1.md (v0.2, 2026-02-15)
- Docs/Design/ArtDirection_v0.1.md (v0.1, 2026-02-14)

기존 사이버 네온 기반 방향은 전부 무효입니다.
다크 판타지/3-4쿼터뷰/마법 이펙트 중심 톤에 맞춰,
MVP 전투 이벤트 중심으로 사운드 디렉션 + AI 제작 파이프라인 + 납품 규격까지 1차안 부탁드립니다.

방향을 잡는 데 필요한 정보가 있으면 바로 질문 주세요.
질문은 우선순위(P0/P1/P2)로 묶어서 보내주시면 빠르게 답변드리겠습니다.
```

## 6. 사운드 디렉터 질문 유도 템플릿

질문이 필요하면 아래 형태로 보내 달라고 요청해 주세요.

```text
[P0] 출시/상업 이용 제약: 사용 가능한 모델/툴 라이선스 범위?
[P0] 플랫폼/성능: 타겟 플랫폼별 오디오 용량 및 동시 재생 제한?
[P1] BGM 운영: 스테이지별 루프 길이/레이어 수(일반-긴장-보스) 확정치?
[P1] 믹스 기준: 마스터 및 버스 라우드니스 목표값?
[P2] 지역화/보이스: 보이스/내레이션 필요 여부?
```
