# Soulspire - Sound Direction First Pass (Agent) v0.2

## 1. 사운드 디렉션 요약

### 톤 키워드

- **Dark Orchestral Minimal**: 소편성(현악 중심 + 저역 악기)으로 무겁되 뭉개지지 않는 공간감.
- **Arcane Resonance**: 마법 공명 계열(벨/공명 금속/잔광 텍스처) 중심.
- **Impact Clarity**: 액션 SFX는 짧고 어택이 선명해야 함.
- **Tension Gradient**: 전투 강도에 따라 레이어가 점진적으로 증가.
- **Earned Reward**: 성장/보상은 화려하지만 과잉은 금지.

### 금지 요소

- 신스Incursion/사이버 네온 계열 음색
- 8bit 칩튠 BGM 중심 운용
- 과도한 보컬/코러스 중심 BGM
- 과도한 잔향(0.8초 초과)으로 타격감 저하
- 급격한 볼륨 스파이크 기반 점프스케어 연출

## 4. BGM 전략

### Sanctum
- 길이: 60-90초 루프
- 레이어: 앰비언스 + 멜로디(2레이어)
- 전환 규칙: 씬 전환 시 1.5초 크로스페이드

### InGame
- 길이: 120초 루프
- 레이어: 베이스 + 퍼커션 + 멜로디(3레이어)
- 전환 규칙: Incursion 시작/종료 시 2.0초 크로스페이드

### Boss
- 길이: 90-120초 루프
- 레이어: 드럼 + 베이스 + 스트링 + 브라스(4레이어)
- 전환 규칙: InGame->Boss 1.0초 크로스페이드, 보스 처치 후 3.0초 페이드아웃

## 5. AI 제작 파이프라인

### 5.1 사운드 생성 도구 비교

| 항목 | Suno AI | Udio |
|------|---------|------|
| 생성 대상 | BGM 중심, SFX는 보조적 | BGM 중심 |
| 프롬프트 제어 | 자연어/태그 제어 | 자연어/태그 제어 |
| 출력 포맷 | MP3/WAV | MP3/WAV |
| 재현성 | seed 미지원(유사 재현) | seed 미지원(유사 재현) |
| 상업 이용 | 유료 플랜 기준 허용(약관 확인 필수) | 유료 플랜 기준 허용(약관 확인 필수) |
| 비고 | BGM 초기 탐색 속도 우수 | BGM 확장 길이 운용 유리 |

### 5.2 선정안

- 선정 도구: **Suno AI**
- 이유 1: BGM 초안 생성 속도가 빠르고 탐색 반복 효율이 높음
- 이유 2: 팀 내 프롬프트 운영 난이도가 낮음
- 이유 3: 상업 이용 기준이 비교적 명확함(최종 약관 재확인 전제)

> 가정: SFX 전용 품질이 부족할 경우 ElevenLabs Sound Effects를 보조 도구로 사용

### 5.3 재현성 로그 포맷 (예시)

```json
{
  "model": "suno-v4",
  "prompt": "dark fantasy orchestral, minor key, tower defense combat loop",
  "seed": "n/a",
  "bpm": 90,
  "key": "D minor",
  "lufs": -14.0,
  "output_path": "Assets/Audio/BGM/BGM_Combat_Wave_01.wav"
}
```

## 6. Unity 납품 규격

### 6.1 오디오 포맷

| 구분 | 포맷 | 샘플레이트 | 비트뎁스 | 비고 |
|------|------|-----------|----------|------|
| BGM | .ogg (Vorbis) | 44,100 Hz | 16-bit | Streaming |
| SFX | .wav | 44,100 Hz | 16-bit | Decompress On Load |
| UI SFX | .wav | 44,100 Hz | 16-bit | Decompress On Load |
| Alert/Jingle | .wav | 44,100 Hz | 16-bit | Compressed In Memory |

### 6.2 라우드니스

| 구분 | Integrated LUFS | True Peak |
|------|-----------------|-----------|
| BGM | -16 LUFS +/- 1 | <= -1.0 dBTP |
| SFX | -12 LUFS +/- 2 | <= -1.0 dBTP |
| UI SFX | -18 LUFS +/- 2 | <= -1.0 dBTP |
| Alert/Jingle | -14 LUFS +/- 1 | <= -1.0 dBTP |

### 6.3 루프 규칙

- BGM: 심리스 루프 필수, 루프 이음새 청취 테스트 통과
- Ambient BGM: 60초 이상 권장
- SFX/UI/Alert: 원샷, 앞뒤 불필요 무음 제거

### 6.4 네이밍 규칙

```text
{Category}_{Context}_{Name}_{Variation}.{ext}
```

예시:
- `BGM_Hub_MainTheme_01.ogg`
- `SFX_Tower_ArrowFire_01.wav`
- `UI_Button_Click_01.wav`
- `ALT_Wave_Start_01.wav`

## 8. 믹스 운영안

### 8.1 Audio Mixer 버스 구조

| 버스 | 부모 | 기본 볼륨(dB) | 역할 |
|---|---|---|---|
| Master | - | 0 | 최종 출력 |
| BGM | Master | -6 | 배경 음악 |
| BGM_Main | BGM | 0 | 메인 루프 |
| BGM_Layer | BGM | -3 | 긴장/보스 레이어 |
| SFX | Master | 0 | 효과음 |
| SFX_Tower | SFX | 0 | 타워 발사/배치/합성 |
| SFX_Enemy | SFX | -2 | 피격/처치/등장 |
| SFX_UI | SFX | -1 | UI/알림 |
| Ambient | Master | -8 | 환경 레이어 |

### 8.2 덕킹 규칙

| 트리거 | 대상 | 감쇠 | Attack | Release | 비고 |
|---|---|---|---|---|---|
| SFX_Tower 재생 | BGM | -4 dB | 20 ms | 300 ms | 타워 피드백 우선 |
| 보스 등장 | BGM_Main | -12 dB | 50 ms | 800 ms | 보스 레이어 강조 |
| SFX_UI 재생 | SFX_Tower/SFX_Enemy | -3 dB | 10 ms | 150 ms | UI 가독성 확보 |
| 일시정지 메뉴 | BGM/SFX/Ambient | -15 dB | 100 ms | 500 ms | 메뉴 집중 |
| Incursion 클리어 팡파레 | BGM | -8 dB | 30 ms | 600 ms | 결과 피드백 강조 |

### 8.3 반복 피로도 체크 기준

- 동일 SFX 재트리거 쿨다운 50ms
- 고빈도 SFX 최소 3변형 라운드로빈
- 동일 버스 동시 재생 상한 8개(초과 시 오래된 인스턴스 페이드아웃)
- 고빈도 SFX 피치 랜덤화 +/-0.05
- 10분 플레이 로그 기준, 특정 SFX 점유율 30% 초과 시 교체/감쇠 후보 지정
